Site para baixar sprites de vários jogos

31 December 2014 on game dev

Descobri o site The Spriters Resource há pouco tempo, e infelizmente não lembro quem compartilhou no Facebook para dar os créditos. Ele é atualizado diariamente com diversas imagens de jogos novos e antigos, o que é perfeito para fazer protótipos de jogos.

Quando comecei o desenvolvimento do Oniro, o jogo que foi o TCC da minha pós-graduação, utilizei sprites de um site semelhante, o Sprite Database. Na época este site não tinha muitas opções de jogos novos, apenas antigos, mas foi muito útil para começar a programar sem ter a arte final do jogo.

UPDATE: Confira estes links enviados pelo FL para baixar modelos e texturas para jogos.

Imagem da capa: Pixel Art Set Of Bandit do Shutterstock.
