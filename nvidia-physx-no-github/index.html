NVIDIA disponibiliza código da PhysX no GitHub

13 March 2015 on game dev

O framework de física da NVIDIA, usado em diversos jogos e engines, agora está com o código publicado no GitHub para que os desenvolvedores possam estudá-lo e até melhorá-lo.

Você pode não saber, mas este framework está implementado em muitas engines, como Unreal e Unity, e até dentro de engines proprietárias como a AnvilNext (Assassin’s Creed) e Dunia (Far Cry) da Ubisoft. Alguns jogos que utilizam esse framework de física: Batman, Bioshock, Daylight, Borderlands, The Witcher, entre muitos outros.

Para acessar o código da PhysX no GitHub siga os procedimentos deste link. São bem parecidos com os procedimentos para acessar o código da Unreal Engine.

Obrigado ao Mestre Kishimoto pela dica!
